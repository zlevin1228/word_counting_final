# Word Counting 4.0

# What is this?

This is a program that will be able to count the amount of times a certain word or phrase appears in a specific file or directory.

# Features

- prints the current working directory, prompting a user to select a specified file in it
- prompts a user to select a word or phrase to look for in a specificed file or directory
- asks a user if the desired results will be case-sensitive or case-insensitive
- prints results through both the program and to a results file.
- includes error handling for blank inputs, invalid files and directories, empty directories, and missing files

# How To Run

1. Open a PowerShell Terminal
2. Navigate to the folder using the cd command
3. Run the program using 'python word_counting4.py'
4. Follow the program

# Functions

- count_words_in_files: Function written in 'word_utils.py'

# Output File

- The results will save to the empty file 'wordcount_results.txt'

# Things to Note

- Any outstanding characters used in the terminal may end the program
